Infinite Monkey Sort
==========

Simple integer sorting algorithm based on infinite monkey theorem.
For details about infinite monkey theorem, see <http://en.wikipedia.org/wiki/Infinite_monkey_theorem>.


Usage
-----

```python
from infinite_monkey_sort import infinite_monkey_sort
print infinite_monkey_sort([1, 0, 3, 2]) // will give [0, 1, 2, 3]
```

The test case is in `test.py`, using [unittest][1].

[1]: https://docs.python.org/2/library/unittest.html "unittest"
